TATANKA STRENGTH V6 - BUG FIX
==============================

Date: November 17, 2024
Issue: Progress and Calendar tabs not functioning

---

PROBLEM IDENTIFIED:
-------------------
The switchTab() function was incorrectly wrapped, causing a JavaScript error that prevented tab switching from working.

Error in code:
```javascript
// This was WRONG:
const originalSwitchTab = switchTab;
function switchTab(tabName) {
    originalSwitchTab(tabName);  // This reference didn't work
    if (tabName === 'progress') {
        displayCustomGoals();
    }
}
```

---

SOLUTION APPLIED:
-----------------
Modified the original switchTab() function directly to include custom goals display:

```javascript
// Fixed version:
function switchTab(tabName) {
    // ... existing tab switching code ...
    
    if (tabName === 'calendar') {
        renderCalendar();
        updateWorkoutLog();
    } else if (tabName === 'progress') {
        updateProgress();
        displayCustomGoals(); // ✅ Added here correctly
    }
}
```
